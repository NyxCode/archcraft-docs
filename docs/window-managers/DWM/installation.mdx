To install archcraft-dwm just run the following command `sudo pacman -S archcraft-dwm`

> DWM is configured by modifying its source code.  If you are not comfortable configuring it this way, DWM might not be the right choice for you.

Its source code is copied to `/opt/dwm-source`. See [Configuration](Configuration) for details.
